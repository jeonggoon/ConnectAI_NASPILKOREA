# NASPILliant B2B 폐쇄형 쇼핑몰 UI/UX 디자인 매뉴얼

## 1. 브랜드 아이덴티티
- **브랜드명:** NASPILliant (나스필리언트)
- **핵심 컨셉:** 미니멀리즘, 아이폰과 같은 심플하고 직관적인 디자인
- **주요 제품군:** 프리미엄 LED 조명 (이클립스 방등, 다기능 센서등, 일체형 손전등 등)

## 2. 컬러 팔레트 & 타이포그래피
- **Primary Color:** #1C1C1E (애플 스타일의 깊은 다크 그레이)
- **Secondary Color:** #F5F5F7 (배경용 깔끔한 오프화이트)
- **Accent Color:** #FFCC00 (조명의 빛을 상징하는 따뜻한 옐로우, 구매 버튼 등에 사용)
- **Font:** Pretendard (세련되고 가독성 높은 산세리프)

## 3. 페이지 구성 (Multiple Pages)
1. **메인 대시보드:** 실시간 재고 현황 요약, 신제품(Weby 센서등 등) 갤러리 슬라이드
2. **B2B 발주 페이지:** 제품 목록, 단가표, 원클릭 대량 주문 버튼
3. **마이페이지:** 주문 내역, 이카운트 ERP 실시간 연동 배송 조회

## 4. 컴포넌트 규칙
- 버튼은 모서리가 둥근 라운드 스퀘어(Border-radius: 12px)로 통일할 것.
- 제품 이미지는 테두리 없이 그림자(Drop-shadow) 효과만 주어 제품 자체가 돋보이게 할 것.
